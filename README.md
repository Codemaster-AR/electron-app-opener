# Electron App Opener
=====================================================

Electron app opener is an experimental project that attempts to open `.exe` files on macOS. Note that this project does not work under all circumstances and is still a work in progress.

### Download
---------

You can download the latest version of Electron App Opener from the [[releases page](https://github.com/Codemaster-AR/electron-app-opener/releases/tag/v1.0.0)(https://github.com/Codemaster-AR/electron-app-opener/releases/tag/v1.0.0). Please note that we only provide the `v1.0.0` release at the moment.
